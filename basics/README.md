# Shell, basics

This directory contains basic shell scripts that perform various tasks:

- `0-current_working_directory`: Prints the current working directory.
- `1-listit`: Lists files in the current directory.
- `2-bring_me_home`: Changes directory to the user's home.
- `3-listfiles`: Lists files in long format.
- `4-listmorefiles`: Lists hidden files.
- `5-listfilesdigitonly`: Lists files with user/group IDs displayed numerically.

Each script starts with `#!/bin/bash` and does exactly one thing as specified in the project.
